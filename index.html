Project 2: Diabetes Prediction using KNN, Random Forest, and Decision Tree

Diabetes is a disease with a rising number of cases, causing 1.5 million deaths globally in 2019. This project compares three classification methods: KNN, Random Forest, and Decision Tree.

Key Features

  • Pregnancies, Glucose, Insulin, BMI, Age as features
  • Target variable: Outcome
  • Model evaluation with/without Min-Max normalization

Model Evaluation

Before Min-Max Normalization

ModelAccuracyComputation Time (s)
KNN74.07%1.5739
Random Forest78.01%4.7255
Decision Tree79.35%3.4833

After Min-Max Normalization

ModelAccuracyComputation Time (s)
KNN77.53%1.5288
Random Forest75.46%4.6285
Decision Tree73.71%3.2686

Conclusion

KNN benefits significantly from Min-Max normalization, while Decision Tree performs better without it. These findings can guide future implementations of diabetes prediction systems.

Project 3: Brazilian E-Commerce (Olist) Dashboard

Key Insights

Dashboard Highlights

  • On-Time vs Late Deliveries

Tools & Techniques

  • Power BI Desktop
  • DAX for calculated measures

The dashboard reveals sales concentration in major cities and a strong on-time delivery rate (93%), providing insights for logistics optimization and regional strategy.
